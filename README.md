# Goal
* 以終為始來看的話，最終我期待你能夠做出一些類似 FooTinder 功能的 API Service (Java, Kotlin, Spring, Redis, PostgreSQL)
* 那開頭的話，我們從 **最基本最基本** 的開始，裝 JVM、裝 IDE
* 我會以 **實戰** 為主，因為我想做的是在迷霧中先幫你開一條直路到目標，中間你想學什麼想繞到哪裡去都行，但我不會多著墨
